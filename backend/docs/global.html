Controller to get the user's profile. This function retrieves the user's data from the database based on the user's ID from the decoded JWT token. It excludes the password field for security reasons.

Controller function to register a new user. This function checks if a user already exists with the provided email, hashes the password, and then creates a new user in the database if the email is unique.
